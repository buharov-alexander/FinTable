# FinTable

Приложение для ведения личных финансов с возможностью отслеживания банковских счетов и управления ими.

## О проекте

FinTable - это веб-приложение для управления личными финансами, созданное с помощью искусственного интеллекта. Приложение позволяет:

- Отслеживать балансы по различным счетам
- Управлять несколькими типами счетов (наличные, депозиты, сберегательные, инвестиционные, криптовалюты)
- Работать с разными валютами (RUB, USD, EUR)
- Редактировать и удалять счета
- Добавлять новые счета

## Технологический стек

- **React** - UI фреймворк
- **TypeScript** - статическая типизация
- **TanStack Query** - управление состоянием и кэшированием данных
- **TanStack React Table** - таблицы с возможностью редактирования
- **Bulma CSS** - CSS фреймворк для стилизации
- **Lucide React** - иконки

## Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск приложения
npm start
```

Приложение будет доступно по адресу `http://localhost:3000`
