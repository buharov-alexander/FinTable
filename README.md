# FinTable

Приложение для ведения личных финансов с возможностью отслеживания банковских счетов и управления ими.

## О проекте

FinTable - это веб-приложение для управления личными финансами, созданное с помощью искусственного интеллекта. Приложение позволяет:

- Отслеживать балансы по различным счетам
- Управлять несколькими типами счетов (наличные, депозиты, сберегательные, инвестиционные, криптовалюты)
- Работать с разными валютами (RUB, USD, EUR)
- Редактировать и удалять счета
- Добавлять новые счета
- Отслеживать актуальные курсы валют в реальном времени

## Технологический стек

- **React** - UI фреймворк
- **TypeScript** - статическая типизация
- **TanStack Query** - управление состоянием и кэшированием данных
- **TanStack React Table** - таблицы с возможностью редактирования
- **Bulma CSS** - CSS фреймворк для стилизации
- **Lucide React** - иконки
- **Axios** - HTTP клиент для запросов к API
- **FreeCurrencyAPI** - источник курсов валют

## Быстрый старт

```bash
# Установка зависимостей
npm install

# Настройка API ключа (опционально)
cp .env.example .env
# Отредактируйте .env файл, добавив ваш API ключ от FreeCurrencyAPI

# Запуск приложения
npm start
```

Приложение будет доступно по адресу `http://localhost:3000`

## Настройка курсов валют

Для получения актуальных курсов валют используется [FreeCurrencyAPI](https://freecurrencyapi.com/):

1. Получите бесплатный API ключ на [freecurrencyapi.com](https://freecurrencyapi.com/)
2. Создайте файл `.env` в корне проекта
3. Добавьте в него: `REACT_APP_CURRENCY_API_KEY=your_api_key_here`

Если API ключ не указан, приложение будет использовать демо-ключ с ограничениями.
